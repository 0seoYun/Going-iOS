## 🌟 Github Convention
- [Feat]: 새로운 기능 구현
- [Fix]: 버그, 오류 해결
- [Add]: Feat 이외의 부수적인 코드 추가, 라이브러리 추가, 새로운 View 생성
- [Del]: 쓸모없는 코드, 주석, 파일 삭제
- [Refactor]: 내부 로직은 변경 하지 않고 기존의 코드를 개선하는 리팩토링 시
- [Move]: 파일 위치 이동
- [Chore]: 그 이외의 잡일/ 버전 코드 수정, 패키지 구조 변경, 파일 이동, 파일이름 변경
- [Docs]: README나 WIKI 등의 문서 개정
- [Setting] : 세팅 관련
